# TriviaQuiz

Quiz app that selects questions randomly from a question pool.

## Link to the page:

https://trivia.ellanan.com/

## Features:

- Randomly generated questions
- Instant feedback on users' input
- Playfully purple palette

## Technologies used

- React
- TypeScript
- Emotion
- HTML
